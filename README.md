# 21days
web wrapper for game "21days" from KolibriOS

запускалка в браузере игры [21 days](https://git.kolibrios.org/nisten/kolibrios/src/branch/main/programs/games/21days) из состава KolibriOS

## компиляция

`make`

для сборки небходимы [emsdk](https://github.com/emscripten-core/) и [typescript](https://github.com/microsoft/TypeScript/)
